### 友链

如需添加友链，请使用以下格式。

```html
<ul class="links">
  <li>
    <a href="//some.link" target="_blank">
      <img src="//path.to/avatar.png">
      <span>Your Friend's Name</span>
    </a>
  </li>
<ul>
```
